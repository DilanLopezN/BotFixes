# Kissbot API

## Description

APP Service API Kissbot.

## Config eg:.
````
export MONGO_URI=mongodb://localhost:27017/kissbot-api
````

## Installation

In project folder, initiate docker image (CONFIG IN PROGRESS)

```
$ ./bin/run_dev
```

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
